# Gym City Website

Welcome to the official customized website for **Gym City** located in Tunis! This repository contains a modernized, feature-rich landing page tailored to represent a high-end gym experience, complete with dynamic functionality and sleek, modern design elements.

## 🚀 Features
- **Modern & Professional Design:** Featuring a customized English-language UI, vibrant layouts, and engaging visual branding with the Gym City logo and genuine facility images.
- **Dynamic Interaction:** Engaging micro-animations to enhance the user experience and create a lively journey through the page.
- **Protein Calculator:** Built-in tool allowing clients to easily calculate their macronutrient needs right from the website.
- **Trainer Profiles & Gallery:** Spotlight showcases highlighting the expert coaching team, complete with galleries of the outdoor spaces, classes, and gym equipment.
- **Testimonials Section:** A dedicated area for authentic client reviews to build trust.
- **Instant Communication:** A floating WhatsApp button provides prospective and current members the ability to quickly get in touch.

## 🛠️ Technology Stack
- **HTML5:** For structured layout and feature integration.
- **Vanilla CSS:** Custom design aesthetics capturing a modern standard without heavy dependency overhead. Includes interactive hover states and rich colors.
- **JavaScript:** Custom scripting to drive the protein calculator logic and other dynamic effects.

## 💻 Getting Started
To view the site locally on your environment:
1. Clone the repository: `git clone https://github.com/khaled-Essmairia/GymWebsite.git`
2. Open `index.html` in your favorite web browser to see it in action! No server necessary for standard viewing.

## 📂 Repository Structure
- `index.html`: The core landing page content and application structure.
- `css/`: Contains styling tokens, custom rules, and foundational visuals.
- `js/`: Scripts bringing the site interactivity, including the protein calculator and animations.
- `images/ & imagess/`: Direct sources for local facility images, logos, client photos, and custom graphics.
